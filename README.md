# breast_cancer_prediction-LogisticR

* Dataset: Breast Cancer Wisconsin, UCI Machine Learning Repository(https://archive.ics.uci.edu/ml/index.php)
* Used Model: Logistic Regression

## Hyperparameter
* learning_rate = 0.003
* batch_size = 30
* iter_num = 12000

## Result
* Trained Epoch: 100000
* Train_acc: 0.967033
* Test_acc: 0.948905
